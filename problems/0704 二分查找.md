# 704. 二分查找



## 题目说明

[Leetcode题目链接](https://leetcode-cn.com/problems/binary-search/)


## 解题思路

```C++
int binary_search(const std::vector<int> &nums, const int &target) {

  auto left = 0;
  auto right = nums.size() - 1;

  while (left <= right)
  {
    auto mid = (left + right) / 2;
    if (nums[mid] == target)
      return mid;
    
    if (nums[mid] < target)
    {
      left = mid + 1;
    }
    else
    {
      right = mid - 1;
    }
  }

  return -1;
}
```
